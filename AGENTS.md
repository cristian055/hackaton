# Agent Instructions

Before creating or changing any UI in this repository:

1. Read `DESIGN.md` at the repo root. It is the source of truth for Comfama
   colors (`#DB0061`, `#303030`, `#CFCFCF`, `#EB003F`, `#0071EB`, `#FFC218`)
   and Roboto typography.
2. Activate `.opencode/skills/comfama-design/SKILL.md` for any UI, design,
   Stitch, or Comfama branding task and follow its contract.

Do not introduce new colors, fonts, or components without updating `DESIGN.md`
first.
